## 项目简介
- 开发框架：Spring Boot 2.7.x + Vue 2.6.x + Element UI
- 数据库：MySQL 8.0+（兼容MySQL 5.7）
- 部署方式：前后端分离，支持单机/集群部署
- 核心特性：用户管理、角色权限、菜单配置、部门管理、字典管理、日志监控等

## 项目介绍
- 技术栈：Vue + Element UI + Spring Boot + MyBatis-Plus
- 核心功能：用户管理、角色权限、菜单管理、系统配置等
- 部署方式：前后端分离

## 快速启动
### 后端启动
1. 配置数据库（MySQL 8.0+）
2. 运行 `RuoYiApplication.java` 启动后端（默认端口：8081）

### 前端启动
1. 进入ruoyi-ui目录，执行 `npm install` 安装依赖
2. 执行 `npm run dev` 启动前端（默认端口：8082）

## 访问地址
- 前端：http://localhost:8082
- 后端接口：http://localhost:8081
- 默认账号：admin / admin123
